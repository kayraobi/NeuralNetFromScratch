# C++ based neural network from scratch


## 🔧 1. Clone the repo
```sh
git clone https://github.com/kayraobi/NeuralNetFromScratch # clone
cd NeuralNetFromScratch
```

## ⚙️ 2. Cmake settings
```sh
mkdir build
cd build
cmake ..
```

## 🛠 3. Compile
```sh
cmake --build .
```

## 🚀 4. Run the program
```sh
.\NeuralNet.exe
```
